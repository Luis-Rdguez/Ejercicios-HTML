HTML

¿Qué es HTML?

HTML, sigla en inglés de HyperText Markup Language (lenguaje de marcas hipertexto), hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referecia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una pagína web, como texto, imágenes, videos, juegos, entre otros.
